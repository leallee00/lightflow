# 快速任务模板

当你想让 AI 快速完成一个小任务，不想走任何流程时，用这个模板。

## 用法

直接说任务，加上"简单任务"标记：

```
[简单任务] 把 src/utils.ts 中的 MAX_RETRY 从 3 改成 5
```

## 什么算简单任务

- 单文件改动
- 纯配置变更
- 不涉及业务逻辑的改动
- 你已经想好了怎么做，只是让 AI 执行

## 效果

Agent 收到 `[简单任务]` 标记后：
- 不加载 Brainstorm 和 Plan
- 直接执行
- 只做轻量 Review

## 常见示例

```
[简单任务] 把这个 HTML 表格改成响应式的
[简单任务] 在 README 加一行安装说明
[简单任务] 把这个 red 变量名改成 primary-color
[简单任务] 把 console.log 删掉
```
